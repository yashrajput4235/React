# React
This project teach you react from basics to advance
Why we use react?
React is a javascrit library for building user interfaces. It is declarative, component-based, and efficient. It is used to build single-page applications, mobile applications, and desktop applications.  
What is the problem of website before react?
Before react, we used to build websites using vanilla javascript. It was very difficult to manage the state of the application. It was also very difficult to maintain the code.
